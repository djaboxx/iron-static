Privacy Policy

Last updated: April 25, 2026

This privacy policy applies to the Iron Static application ("App"), operated by Dave Arnold, for the purpose of publishing music content to Instagram on behalf of the Iron Static band account.

1. What the App Does

The App is an internal tool used solely by Iron Static to automate the posting of promotional images and captions to the band's Instagram account via the Meta Graph API. It is not a public-facing consumer application.

2. Data We Collect

The App does not collect, store, or process personal data from any end users. The only data processed is:

  • Instagram access tokens used to authenticate API requests (stored as environment variables, never logged or transmitted to third parties)
  • Band-owned image files and caption text generated by the band for posting

3. Data Sharing

We do not share, sell, rent, or otherwise disclose any data to third parties. Image files are temporarily staged in Google Cloud Storage solely to satisfy the Meta Graph API's requirement for a publicly accessible image URL. These files are the band's own original content.

4. Third-Party Services

The App interacts with:

  • Meta Graph API — for Instagram content publishing. Governed by Meta's own Privacy Policy.
  • Google Cloud Storage — for temporary image hosting. Governed by Google's Privacy Policy.

5. Data Retention

No user data is retained. Access tokens are stored locally in environment variables and are not written to any database or log. Staged images in Google Cloud Storage are managed by the band and can be deleted at any time.

6. Contact

For questions about this policy, contact: dave@roknsound.com
